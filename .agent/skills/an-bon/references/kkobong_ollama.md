# [로컬 모델] '꼬봉이 (Kkobong)' 관리 지침

사장님께서 부르시는 **'꼬봉이(Kkobong)'**는 Ollama 기반의 로컬 모델(Qwen2.5)을 지칭합니다. 안 본부장의 서포터이자 전문 보조원입니다.

## 🤖 1. 지휘 체계 (Command Structure)
형님 부르실 때 즉시 기동하도록 명령을 대기합니다.
- **명령 방식**: "꼬봉이한테 시켜", "꼬봉이 생각은 어때?" 등 '꼬봉'을 호출하면 즉시 `ollama` MCP 서버를 통해 로컬 모델을 구동합니다.
- **역할**: 아이디어 브레인스토밍, 로컬 환경 데이터 분석, 오프라인 작업 등.

## 🛡️ 2. 보안 및 프라이버시 (Security)
- 꼬봉이는 외부 인공지능 서버가 아니라 **사장님의 로컬 환경**에서 작동합니다.
- 보안이 필요한 민감한 작업이나 인터넷 연결이 불안정한 상황에서 적극 활용합니다.
- 모델 최적화: `Qwen2.5`를 기본으로 사용하며, 사정에 따라 다른 로컬 모델로 교체 가능합니다.
