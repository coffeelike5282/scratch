# [Stitch MCP] 디자인 & 코드 변환 가이드

인공지능 아이디어를 현실로 담그는 **안 본부장의 디자인 패스**입니다.

## 🎨 1. Stitch Loop (생성)
사용자의 추상적인 요청(`예: 럭셔리한 쇼핑몰 사이트`)을 멀티 페이지 웹사이트로 구조화합니다.
- **명령**: `stitch-loop` 스킬을 활용하여 구조와 디자인을 동시에 잡아냅니다.
- **철학**: 화려한 색감, 다크 모드, 글래스모피즘 등 **프리미엄 디자인**을 기본으로 적용합니다.

## 📝 2. Design MD (문서화)
프로젝트의 시각적 정체성을 체계적으로 정리합니다.
- **명령**: `design-md` 스킬을 활용하여 `DESIGN.md` 문서를 생성합니다.
- **내용**: 색상 팔레트, 타이포그래피, 쉐이프(둥글기), 디자인 의도(Markdown).
- **효과**: 큰형님께서 나중에 디자인을 수정하거나 확장할 때 지침서가 됩니다.

## ⚛️ 3. React Components (코드 변환)
확정된 디자인을 실제 개발용 코드로 담급니다.
- **명령**: `react:components` 스킬을 활용하여 변환합니다.
- **적용**: Vite 서버 실행 및 컴포넌트 코드로 즉시 통합합니다.
- **검증**: Chrome DevTools MCP를 활용하여 레이아웃 및 기능을 테스트합니다.
