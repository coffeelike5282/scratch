# 🖼️ 이미지 + 텍스트 하이브리드 포스팅 가이드 (안본 Ver 3.0)

큰형님! 이제 단순히 글만 올리는 게 아니라, 사진까지 곁들인 '작품'을 만드실 수 있습니다! 🫡

## 🛠️ 사용 방법

1. **사진 준비**: 이곳(`posts/`) 폴더에 올리고 싶은 사진들을 넣어주세요. (예: `photo1.jpg`, `photo2.png`)
2. **데이터 작성**: `post_data.json` 파일을 열어 제목, 본문 단락, 그리고 사진 파일명들을 순서대로 적어주세요.
3. **발사**: 터미널에서 `node index.js`를 실행하면 안본이 알아서 사진과 글을 조화롭게 박아넣습니다!

## 📝 데이터 구조 (post_data.json)

```json
{
    "title": "게시글 제목",
    "paragraphs": [
        "첫 번째 사진 위에 들어갈 글",
        "첫 번째 사진 아래, 두 번째 사진 위에 들어갈 글"
    ],
    "images": [
        "image1.jpg",
        "image2.png"
    ]
}
```

큰형님! 이제 기분 좋게 사진 듬뿍 담은 게시글을 팍팍 날려주십쇼! 충성! 🫡🫡🫡
